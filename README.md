# Spotify_Data_Analysis

## Overview
This project is a data analysis project focused on exploring Spotify music data to identify trends and patterns in songs, artists, popularity, and music characteristics.

The goal of this analysis was to clean the dataset, explore the data, and create visualizations to better understand factors that influence song popularity.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset
The dataset contains information about Spotify tracks, including:
- Track name
- Artist
- Genre
- Popularity
- Track features
- Explicit content information

## Analysis Performed
- Data cleaning and preparation
- Checked for missing values and duplicates
- Explored song and artist trends
- Analyzed popularity patterns
- Created data visualizations

## Visualizations
The project includes charts showing:
- Top artists
- Most popular songs
- Popularity trends
- Other Spotify music insights

## Files
- Spotify_Analysis.ipynb - Main analysis notebook
- spotify_data clean.csv - Cleaned dataset

## Conclusion
This project helped explore Spotify data and demonstrate the process of cleaning, analyzing, and visualizing real-world datasets using Python.
